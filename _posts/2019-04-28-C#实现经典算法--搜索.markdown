# 搜索算法
## 广度优先搜索
<li>优先一层一层遍历，每层遍历都以上一层遍历的结果作为起点，遍历一个距离所能访问到的所有节点。遍历过的节点不能再次遍历。</li>
<li>可以求解无权图的最短路径长度。因为第一次遍历到目的节点，其所经过的路径为最短路径。</li>
